# Mastering-React
    Understand React inside out 
    Boost career prospects 


## Requirements 

    - nvm
        - https://github.com/nvm-sh/nvm
    - npm


### Steps
Install package:
    
    npm i -g create-react-app 
        - adds react package

To create an app 

    create-react-app react-app
        - this command actually creates application

Run:

    npm start 
        - has to be in root
